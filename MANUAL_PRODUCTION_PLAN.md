# FANUC ROBODRILL 기초과정 교재 제작 설계

## 1. 제작 목표

FANUC ROBODRILL 초보자 교육 교재를 총 20개 항목, 약 200-250페이지 규모로 제작한다. 각 항목은 독립 수업 단위로 읽히면서도 전체 과정에서는 장비 이해, 전원 조작, 원점복귀, 수동 이동, 좌표계, MDI, 프로그램 호출, 공구교환, 오프셋, 첫 가공 실습으로 자연스럽게 이어지도록 구성한다.

## 2. 전체 산출물

| 산출물 | 위치 | 설명 |
| --- | --- | --- |
| 항목별 원고 | `src/content/level1_itemXX.md` | 각 항목의 본문, 표, 실습, 문제 |
| 이미지 자산 | `src/assets/images/` | 실제 ROBODRILL 사진, 패널 사진, 현장 예시 사진 |
| 도해 자산 | `src/assets/diagrams/` | 좌표계, 작업 흐름, 안전 점검, 프로그램 흐름도 |
| 이미지 출처 | `src/assets/images/ATTRIBUTION.md` | 웹 이미지 출처, 작성자, 라이선스, 사용권 주의사항 |
| HTML 출력 | `output/html/` | 검수용 중간 산출물 |
| PDF 출력 | `output/pdf/` | 항목별 PDF와 통합 PDF |

## 4. 항목별 표준 구성

각 항목은 아래 순서를 기본 템플릿으로 사용한다. 항목 성격에 따라 일부 절을 합치거나 확장할 수 있다.

| 순서 | 섹션 | 페이지 목표 | 내용 |
| --- | --- | ---: | --- |
| 1 | 표지 또는 항목 시작 | 1 | 제목, 학습 대상, 선행 지식 |

## 5. 항목 02 예시 설계

### 02 전원 ON/OFF

권장 분량: 10-14페이지

| 섹션 | 내용 |
| --- | --- |
| 교육 목표 | 전원 투입 전 확인, CNC 전원 ON, Servo ON, 안전한 OFF 절차를 설명하고 수행한다 |
| 안전사항 | 비상정지 상태, 도어 상태, 에어 압력, 이전 작업 상태, 미완료 알람 확인 |
| 패널 사진 | 실제 CNC 패널, 전원 버튼, Servo ON/OFF 관련 위치 |

| Servo ON | Servo ON의 의미, 축이 움직일 수 있는 상태가 되는 시점 |
| Home Return | 전원 투입 후 기준점 복귀와 연결되는 이유 |
| 전원 OFF | 축 정지, 주축 정지, 프로그램 정지, CNC 종료, 주 전원 차단 |
| 실습 | 지도자 감독하에 전원 투입 전 점검표 작성, 화면 상태 읽기 |
| 시험 문제 | 순서 배열, 위험 상황 판단, 단답형 보고 문장 작성 |

## 6. 이미지 제작 원칙

이미지는 두 종류로 나누어 사용한다.

1. 실제 이미지
   - 실제 ROBODRILL 장비, CNC 패널, 공구교환기, 작업 영역 사진
   - 가능하면 공식 제조사 이미지 또는 라이선스가 명확한 공개 이미지를 사용한다.
   - 출처, 작성자, 라이선스, 원본 URL을 `ATTRIBUTION.md`에 기록한다.

2. 설명용 이미지
   - 축 방향, 전원 순서, 원점복귀 흐름, 프로그램 실행 흐름 등
   - 직접 제작한 SVG 도해를 우선 사용한다.
   - 버튼 위치를 설명할 때는 실제 사진 위에 번호를 붙인 주석 이미지를 별도 제작한다.

공식 제조사 이미지는 교육 내부용 초안에는 사용할 수 있으나, 외부 배포용 PDF에는 사용 허가를 확인한다.

## 7. 제작 워크플로우

각 항목은 다음 반복 절차로 제작한다.

1. 항목 기획
   - 학습 목표 5-8개 작성
   - 필요한 사진과 도해 목록 작성
   - 실습 장비 조건과 금지 조작 정리

2. 이미지 수집
   - 웹 실사 이미지 후보 검색
   - 공식 이미지와 공개 라이선스 이미지 구분
   - 로컬 저장 후 `ATTRIBUTION.md` 업데이트

3. 원고 작성
   - `src/content/level1_itemXX.md` 작성
   - 표, 안전 박스, 실습 박스, 문제 포함
   - 초보자 기준으로 버튼 조작 전 확인 문장을 반복 삽입

4. HTML/PDF 생성
   - 항목별 HTML 생성
   - 항목별 PDF 생성
   - 전체 PDF 병합은 20개 항목 초안이 갖춰진 뒤 진행

5. 검수
   - 페이지 수 확인
   - 한글 깨짐 확인
   - 이미지 출처 확인
   - 표와 캡션 겹침 확인
   - 실습 절차가 위험하거나 모호하지 않은지 확인

6. 수정
   - 페이지가 부족하면 실습, 예시, 오류 사례를 추가한다.
   - 페이지가 넘치면 반복 설명을 줄이고 표로 압축한다.
   - 이미지가 너무 크면 CSS 이미지 최대 높이를 조정한다.

## 9. 품질 기준

완료된 항목은 다음 기준을 만족해야 한다.

- 10-20페이지 범위 안에 있다.
- 실제 이미지 또는 설명용 이미지가 최소 2개 이상 포함된다.
- 안전사항이 별도 박스 또는 표로 표시된다.
- 단계별 절차가 번호 목록으로 제공된다.
- 실습 과제가 최소 1개 이상 있다.
- 시험 문제가 객관식, 단답형, 실습 평가 중 2종 이상 포함된다.
- 이미지 출처가 누락되지 않는다.
- PDF에서 한글이 깨지지 않는다.
- 표와 이미지가 페이지 밖으로 넘치지 않는다.

## 10. 권장 제작 순서

처음부터 20개 항목을 모두 쓰기보다, 다음 순서로 품질 기준을 고정한다.

1. 01-03 작성: 구조, 전원, 원점복귀의 기본 스타일 확정
2. 04-06 작성: 수동 이동과 좌표계 설명 방식 확정
3. 07-10 작성: 패널 조작, 프로그램, 공구, 오프셋 설명 방식 확정
4. 11-15 작성: 세팅과 가공 준비 절차 확장
5. 16-20 작성: 점검, 품질, 기록, 첫 가공 통합 실습 완성
6. 전체 용어 통일, 그림 번호 정리, 페이지 번호와 목차 구성
7. 통합 PDF 생성 및 최종 검수
